✎ 1. Modifier le contenu
Bas de page → Administration → email + mot de passe Firebase
Textes encadrés en rouge = modifiables. + ajouter, ✕ supprimer.
☁ 2. Sync cloud
Mode édition → ☁ Sync cloud → URL Firebase + Clé API → Connecter.
Sauvegarde auto sur tous les appareils.
📷 3. Image
imgur.com → uploader → clic droit → Copier l'adresse → coller dans le champ Image.
🎥 4. Vidéo YouTube
YouTube → Partager → Copier le lien → coller dans le champ Vidéo.
📄 5. Document PDF
Google Drive → lien de partage public → coller dans le champ Lien.
